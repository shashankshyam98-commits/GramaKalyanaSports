\## 📌 Overview



GramaKalyanaSports is a smart Android application built to digitize rural sports tournaments and improve the experience for organizers, scorers, players, and fans.



The app supports live score tracking, tournament registration, multiple sports dashboards, and AI-generated match summaries.

\## ✨ Features



\### 👤 User Features

\- Secure Login Authentication

\- Fan Dashboard

\- Profile Management

\- Live Match Viewing

\- Deep Link Support



\### 🏅 Tournament Management

\- Admin Dashboard

\- Share Match Insights
